# **"Автоматизированная система генерации XML-документов на основе языковых моделей и методов объяснимого искусственного интелекта в строительной области"**

!!! ***Тема все еще обсуждается с руководителем***

Описание:
Выпускная квалификационная работа посвящена разработке автоматизированной системы генерации XML-документов на основе языковых моделей и методов объяснимого искусственного интеллекта (XAI) для применения в строительной отрасли.

В рамках работы рассматривается использование больших языковых моделей (LLM) для автоматического формирования структурированных XML-документов по текстовым описаниям строительных объектов, проектных решений и технических характеристик. Предлагаемая система обеспечивает преобразование естественно-языковых данных в машиночитаемый формат, соответствующий установленным XML-схемам, используемым при информационном моделировании зданий и сооружений (BIM) и обмене данными с государственными информационными системами (например, ГИСОГД).

Для повышения прозрачности и доверия к результатам работы модели в систему интегрированы методы объяснимого искусственного интеллекта. Они позволяют визуализировать вклад отдельных фрагментов исходного текста в формирование конкретных элементов XML-документа, выявлять потенциальные ошибки и отклонения от схемы, а также объяснять выбор структуры и атрибутов.

Результатом работы является программный прототип, реализованный в виде интерактивного веб-приложения (или Jupyter Notebook), обеспечивающего ввод текстового описания строительного объекта, генерацию XML-документа и визуализацию объяснений, полученных с помощью XAI-модуля. Проведено экспериментальное исследование точности генерации, корректности соответствия схемам и интерпретируемости решений модели.
