# **"Исследование и визуализирование источников галлюцинаций больших языковых моделей с использованием методов объяснимого ИИ"**

#! ***Тема все еще обсуждается с руководителем***

Описание:
Работа направлена на исследование и сравнение методов объяснимого искусственного интеллекта (XAI) а так же разработку гибридного метода и визуализацию результатов, применимых к большим языковым моделям. Основное внимание уделяется выявлению и интерпретации источников галлюцинаций. Предлагается гибридный метод интерпретации, основанный на комбинации существующих XAI-подходов. В качестве продукта создаётся прототип, позволяющий вводить текстовые запросы, получать ответы моделей и визуализировать влияние входных токенов на генерируемый текст.
