# **"Разработка и исследование методов объяснимого искусственного интеллекта для анализа галлюцинаций в больших языковых моделях"**

!!! ***Тема все еще обсуждается с руководителем***

Описание:
В работе проводится исследование проблемы галлюцинаций в больших языковых моделях (LLM) и методов их интерпретации с помощью подходов объяснимого искусственного интеллекта (XAI). Планируется провести сравнительный анализ существующих XAI-методов (например, attention-based, LIME, SHAP, Integrated Gradients и др.) на небольших открытых языковых моделях. На основе выявленных особенностей будет предложен гибридный метод объяснимости, ориентированный на анализ влияния входных токенов на появление галлюцинаций в ответах модели.
Практическим результатом станет прототип системы (Jupyter Notebook или веб-приложение), позволяющей загрузить промпт, получить ответ модели и визуализировать источники галлюцинаций через подсветку значимых токенов. Итогом работы станет выявление закономерностей возникновения галлюцинаций и формулировка рекомендаций по интерпретации LLM.
