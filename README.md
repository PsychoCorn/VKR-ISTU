# VKR-ISTU

*"Исследовать и визуализировать источники галлюцинаций больших языковых моделей с использованием методов объяснимого ИИ"*

Кондратенко Артём Кириллович АСУб-22-1

Описание:
Работа направлена на исследование и сравнение методов объяснимого искусственного интеллекта (XAI), применимых к большим языковым моделям. Основное внимание уделяется выявлению и интерпретации источников галлюцинаций. Предлагается гибридный метод интерпретации, основанный на комбинации существующих XAI-подходов. В качестве продукта создаётся прототип, позволяющий вводить текстовые запросы, получать ответы моделей и визуализировать влияние входных токенов на генерируемый текст.
